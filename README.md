## Dev Setup

First, clone the repo via git and install dependencies:

```bash
git clone https://github.com/rassweiler/robot-project-utility.git your-project-name

cd your-project-name

yarn
```

## Dev Start

```bash
yarn start
```

## Packaging (Local Platform)

```bash
yarn package
```
